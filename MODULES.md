## Doc structure (every module)

1. **Theme teaser** — atmosphere in a few lines  
2. **Quick install** — `@import` URL, host map, `@link` / `@create` portal, `go` commands  
3. **What to expect** — features, hidden objects, puzzles teased (no solutions)  
4. **Extension ideas** — optional builder hooks
